Real-Time Treasury Compliance for DAOs

The Problem

DAO treasuries hold billions in crypto assets governed by risk policies that nobody enforces in real-time. Compliance is checked manually — if it's checked at all. Breaches go unnoticed until a governance post-mortem.

The Solution

AEGIS monitors Gnosis Safe treasuries against a YAML policy file and flags violations instantly. Define your rules once, get continuous compliance reports and Telegram alerts — no smart contract changes required.

5 Policy Rule Types

allocation_cap

No single token exceeds X% of total portfolio value

stablecoin_floor

Stablecoins must remain above a minimum % of holdings

single_asset_cap

Cap absolute USD exposure to any single asset

max_tx_size

Block or flag transactions above a USD threshold

inactivity_alert

Alert if no Safe activity for N hours/days
